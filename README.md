# Blob Counter
## Details
This program counts the number of blobs on the screen based of a threshold value in RGB.

## Setup
open up a commandline and run:
```
pip3 install cv2
pip3 install numpy
pip3 install tk
pip3 install pillow
```

## Run Counter
Run:
```
python3 counter.py
```

It will automatically look for pictures inside the folder "<repo>/input_images/"
